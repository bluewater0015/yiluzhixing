## 蓝猫教育官网

### 导入项目

	$ git clone https://github.com/liwudi/BlueCatWebsite.git

### 安装依赖

	$ cd BlueCatWebsite
	$ npm install & bower install

### 启动服务

	$ gulp serve

### 项目打包部署

	$ gulp

### 部署git地址
  https://github.com/liwudi/BlueCatWebsite.git
